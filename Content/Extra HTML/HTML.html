HTML

Structure

  • <html> contains all other HTML elements in the document
  • <head> contains descriptive elements about the document
  • <body> contains content to be displayed by the browser
  • <meta> declares the character encoding is UTF-8
  • <link> specifies external css file
  • <script> specifies external JavaScript file

Elements

  • <p> container for text and other HTML elements
  • <div> container element used for logical grouping of content
  • <a> anchor element used when adding links to a page

Special Character Entities

  • Can get special symbols by typing "&" followed by one of the following strings:
  • nbsp; nonbreakable space  
  • lt; less than symbol <
  • gt; greater than symbol >
  • copy; copyright symbol ©
  • trade; trademark symbol ™

List Elements

  • <ul> unordered list
  • <ol> ordered list
  • <dl> description list
  • <li> list item

Tables

  • <table> defines a table
  • <tr> defines a table row
  • <th> defines a table header
  • <td> defines table data/cell
  • <caption> adds a caption to a table
  • To make a cell span more than one column use the attribute colspan
  • To make a cell span more than one row use the attribute rowspan

Semantic Structure Elements

  • <header> contains the documents heading
  • <main> contains the main unique content of the document
  • <section> broad element for larger sections of content
  • <article> used for blocks of content that can be read
  • <aside> used for sidebars or grouping of nonessential elements like ads
