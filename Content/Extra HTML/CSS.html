CSS

CSS Styles

Inline styles

  • Style rules placed within an HTML element via the style attribute.
  • Example: <h2 style="font-size: 24pt;"> Description </h2>

Embedded style sheet

  • Style rules placed in a style tag inside the head tag.
  • Example: <style> h2 { font-size: 24pt; } </style>

External style sheet

  • Style rules placed within and external text file with the .css extension.
  • External style sheets are implemented using the link tag.
  • Example: <link rel="stylesheet" href="styles.css"/>

Selectors

Element Selectors

  • Select all instances of a give HTML element.
  • Example: h2 { font-size: 24pt; }

Class Selector

  • Allows you to simultaneously target different HTML elements regardless of their position in the document.
  • Example:
  • <style> .first { font-size: 24pt; } </style>
  • <p class="first">Reviews</p>

Id Selector

  • Allows you to target a specific element by an id attribute regardless of type or position.
  • Example:
  • <style> #first { font-size: 24pt; } </style>
  • <p id="first">Reviews</p>

Attribute Selector

  • Provides a way to select HTML elements either by the presence of an element attribute or value of an attribute.
  • Example:
  • <style> [title] { font-size: 24pt; } </style>    This will target all elements with a 'title' attribute.

Pseudo Selectors

  • Pseudo-element selector: Select something that does not exist explicitly as an element in the HTML document by still a recognizeable object.
  • Pseudo-class selector: Applies to an HTML element, but targets either a particular state, or a variety of family relationships.
  • Pseudo-element and Pseudo-class selectors:
    • a:link
    • a:visited
    • :focus
    • :hover
    • :active
    • :checked
    • ::first-child
    • ::first-letter
    • ::first-line
